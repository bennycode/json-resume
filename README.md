# JSON Resume Builder

A web-based UI for creating resumes that follow the [JSON Resume](https://jsonresume.org/schema) standard. Built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Form-based editing for the JSON Resume `basics` section (personal info, location, social profiles)
- Live JSON preview as you type
- Empty fields are automatically stripped from the output
- Download your `resume.json` with one click
- Dark mode support

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start building your resume.
